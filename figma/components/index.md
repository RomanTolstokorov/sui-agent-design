## How to use componentKeys

## Related Local Rules

Use `figma/foundations/typography.md` and `figma/foundations/typography.json` for typography styles, font families, text style keys, and font-loading requirements.

Use `figma/foundations/colors.md` and `figma/foundations/colors.json` for Light/Dark palette tokens, surfaces, and elevation style keys.

Bind fills and strokes to the published color variables by `variableKey`. Do not use raw hex/RGB colors in Figma mockups unless a needed token does not exist yet; if that happens, call it out as a token gap.

Application screen composition rules live in `app/rules.md`.

Use `figma/foundations/layout.md` and `figma/foundations/layout.json` for spacing and corner radius variables.

Do not query Figma for typography, color/elevation, spacing, or shape tokens during routine mockup work unless the design system has changed.
